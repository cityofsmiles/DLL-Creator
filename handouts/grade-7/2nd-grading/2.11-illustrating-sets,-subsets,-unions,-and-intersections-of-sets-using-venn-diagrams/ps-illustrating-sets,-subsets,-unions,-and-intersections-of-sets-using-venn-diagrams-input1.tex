Consider \(U = \{x|x \text{ is a whole number less than } 10\}\). Let \(A = \{1,3,5,7,9\}\) and \(B = \{2,3,4,5,6\}\). Find the following:
\begin{multicols}{2}
\setlist{nolistsep}
\begin{enumerate}[noitemsep, label = \color{blue}\arabic*. ]
\item \(A'\)
\item \(B'\)
\item \(A \cup B\)
\item \(A \cap B\)
\item \(A \cap U\)
\item \(B \cap U\)
\item \(A - B\)
\item \(B - A\)
\item \((A \cap B)'\)
\item \(A' \cap B'\)
\end{enumerate}
\end{multicols}
