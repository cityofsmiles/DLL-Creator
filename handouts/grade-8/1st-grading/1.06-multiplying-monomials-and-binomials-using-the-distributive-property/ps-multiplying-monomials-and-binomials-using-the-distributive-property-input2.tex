Simplify each of the following expressions by using the distributive property to multiply the monomials, binomials, or multinomials.
\begin{multicols}{2}
\begin{enumerate}
    \item \( 2x(3x + 4) \)
    \item \( 5y(4y - 6) \)
    \item \( 3a(a + 2b) \)
    \item \( 6m(3m - 5) \)
    \item \( (x + 3)(x + 4) \)
    \item \( (2y - 5)(y + 3) \)
    \item \( (3a + 4)(a + 2b) \)
    \item \( (5x - 2)(3x + 6) \)
    \item \( (x + 2)(x^2 + 3x + 1) \)
    \item \( (3y - 4)(y^2 + 5y + 2) \)
\end{enumerate}
\end{multicols}
