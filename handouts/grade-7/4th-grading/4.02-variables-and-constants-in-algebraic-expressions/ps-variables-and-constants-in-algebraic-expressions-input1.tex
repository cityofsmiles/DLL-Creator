Identify the variables and constants in the following algebraic expressions.

    \item $2y - 7$

    \item $5x^2 - 4x + 1$  
    \item $k - 3j + 9$  
    \item $\frac{1}{2}x + 3y - 4z + 7$  
    \item $7a^2 - 2b^2 + 3c - 5$  
    \item $x^3 - 4x + 2y - 9$  
\end{enumerate}
\end{multicols}
